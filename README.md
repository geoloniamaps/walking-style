# walking-style

歩行時に見やすい地図スタイル (Map style optimized for walking/pedestrian use)

暖色系の親しみやすい配色で、歩道・遊歩道・階段を強調し、建物形状やランドマークを適度に表示するスタイルです。

## プレビュー

https://geoloniamaps.github.io/walking-style/

## 使い方

```
https://geoloniamaps.github.io/walking-style/style.json
```

## 開発

```bash
npm install
npm start
```

## ビルド

```bash
npm run build
```

`docs/style.json` が生成されます。GitHub Pages で配信されます。
